ПРИМЕЧАНИЕ: При отсутствии предварительных знаний в программировании настоятельно рекомендую начать со Scratch и сделать несколько проектов и игр на нём. Смотрите griffpatch на YouTube. После того как освоитесь в Scratch, учить C будет намного проще.

## Ресурсы
- https://en.cppreference.com/c (продвинутый уровень)

## План
1. Настройка: рекомендую `gcc` (или `clang`) + VSCode. Ещё один хороший вариант — `zig cc`. Настоятельно рекомендуется Linux. Также установите `make` и, возможно, `pkgconf`.

2. "Hello, World!" (используйте одну команду для сборки). `%d` и `%f` в `printf`. Базовые математические операторы, некоторые полезные функции из stdlib, например `sqrt`, `pow`.

3. Переменные (тип `int`). `scanf` (но пока без указателей). Всегда инициализируйте переменные, неинициализированные переменные при сильной оптимизации могут привести к неопределённому поведению (UB)! Напишите несколько простых программ (например, решатели уравнений: линейных, квадратных, биквадратных).

4. `if`-`else` и сравнения. Добавьте проверку ошибок в эти решатели. Сделайте простой калькулятор (ввод типа `char` для операций и цепочки `if`-`else if`-`else`).

5. Цикл `while`. Сделайте так, чтобы он выводил что-то заданное количество раз без использования `for`, затем представьте цикл `for` как сокращение. (Это тяжёлая часть...)

6. Тип `bool`, истинные и ложные значения других типов (`while (n) n--;`).
   Введите константные переменные. Они помогут отлавливать ошибки вида `if (var = 5)`.
   Также можете познакомить с другими функциями stdlib, например `rand`.
   Сделайте несколько базовых игр, например "Угадай число". Если вы любите сложности, сделайте крестики-нолики без массивов (9 переменных для поля и много `if`'ов).

6. Массивы, статические и с переменным размером на стеке. Перепишите крестики-нолики с массивами. Не забывайте проверять границы: хотя выход за пределы массива может быть незаметен, это неопределённое поведение (UB)!
   Объедините массивы с циклами. Сделайте Wordle, но с числами.

7. Строки (символьные массивы с нулевым завершением). Символьные литералы. Некоторые строковые функции (например, `strlen`, `strchr`). Упомяните указатели (`const char *`). Сделайте несколько текстовых игр, например "Виселица".

7. Функции, параметры, оператор `return`. Сделайте несколько простых функций преобразования. Сделайте больше игр (например, "Четыре в ряд"), покажите, насколько полезны функции.

7. Прочитайте немного простого кода на C. Я не нашёл хороших примеров из реального мира, поэтому мне пришлось переписать немного моего C++ в более простой C. Введите размеры типов (биты, байты и слова, `int16_t`, `size_t`).

8. Сделайте консольный игровой движок (V1). Используйте одномерный массив char и формулу `y * width + x`. Дайте представление о плоской модели памяти. Сделайте простую игру "Змейка" с его помощью (доска + координаты головы и хвоста хранят всё состояние игры).

9. Указатели. Консольный игровой движок (V2).
